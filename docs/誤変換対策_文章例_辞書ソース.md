# 誤変換対策: 文章例と辞書ソース

最終更新: 2026-02-14

## 1) 変換確認に使える文章例（長文向け）

- 音声入力の精度が上がってきたと思うけど、まだ制度とか漢字の変換で迷うことがある
- これからローソンでタバコ買うの？
- まゆはこれから新宿で合流する予定
- ありがとう、あとで連絡する
- 〇〇インターを降りたら、次は△△配送センターに向かう
- 今日は渋滞がひどいから到着が30分くらい遅れそう
- この変換候補は文脈に合ってる？

## 2) 既にアプリで自動更新している辞書

- SKK辞書群（一般語/固有名詞/人名/駅名/地名など）
  - https://skk-dev.github.io/dict/
- 日本郵便 郵便番号データ（住所系の語を補強）
  - https://www.post.japanpost.jp/zipcode/download.html
  - 直接ダウンロード例: https://www.post.japanpost.jp/zipcode/dl/utf/zip/utf_ken_all.zip

## 3) 追加検討できる無料ソース（任意）

- Wikipedia タイトル一覧ダンプ（新語・作品名・固有名詞の補強）
  - https://dumps.wikimedia.org/jawiki/latest/
  - 例: jawiki-latest-all-titles-in-ns0.gz
- Sudachi辞書（形態素辞書。直接IME辞書としては変換処理が必要）
  - https://github.com/WorksApplications/SudachiDict

## 4) 運用メモ（効く順）

- 1. 単語帳に「誤変換しやすい語」を word + readingKana で追加
- 2. 辞書メンテ画面のワンタップ更新を定期実行
- 3. 長文で崩れる語は、まず短いフレーズで単語帳登録してから再テスト
